## Списки

Чтобы добавить не нумерованные списки, необходимо добавить (*). Например:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты пронумеровать. Например:
1. Элемент
2. Элемент
3. Элемент
